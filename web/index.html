关于 PPTTER

零信任匿名聊天：服务器只转发不可解密的密文，不落库、不写盘、不记录 IP 或内容。

  • 端到端加密：X25519 ECDH + HKDF（前向保密）、Ed25519 逐条签名（发送者认证）。
  • 防重放、长度分桶填充、密钥指纹可做带外校验。
  • 私聊支持 WebRTC P2P 直连，大文件不经服务器中转（群聊不建立 P2P，避免 N² 连接）。
  • 纯本地静态资源，无任何第三方 CDN；UI 使用 Alpine.js CSP 构建，兼容严格 CSP。
安全提示：端到端加密保护的是「传输中」与「服务器」，但你的浏览器、操作系统和输入法能在加密前看到明文。请使用可信浏览器、无痕窗口、不装可疑扩展，并尽量用离线输入法（关闭云联想），避免击键被上传。
